A text-based survival game built to practice **Java Object-Oriented Programming (OOP)**.

### Concepts Used:
* **Interfaces**: Defining a contract for all characters.
* **Abstract Classes**: Reducing code repetition for shared attributes (Name, HP).
* **Polymorphism**: Handling different character types (Human, Alien) using a single interface.
* **Game Loop**: Using `while` loops and `Random` logic for dynamic gameplay.
